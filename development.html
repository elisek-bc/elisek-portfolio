Kattenhotel

html / bootstrap / css

Een kattenhotel die al jarenlang dezelfde website gebruikte had dringend nood aan een frissere look & feel. Ik paste het logo aan en maakte op basis daarvan 3 webpagina’s die als voorbeeld konden dienen voor de rest van de website. De kleuren werden uit mijn selectie aan mogelijkheden gekozen door de klant, in dit geval mijn docent.

Restaurantwebsite

html / bootstrap / css / git

Project in team waarbij zelf een restaurant werd bedacht en de website hiervan uitgewerkt in een onepager. Elk onderdeel werd uitgewerkt door een ander lid van het team. Het ontstaan van Mario & Luigi’s tratoria was een feit. De site richt zich naar een jong en dynamisch publiek, maar wil toch ook de Italiaanse roots van de broers in de verf zetten.
